Problem

This website also came with a few problems along the way toward completion. As you can see, I put much detail into this site while also using a lot more creative displays. The problem while building this site would be, well, just that! It has a lot of things going on, and trying to figure out the best way to make it aesthetically pleasing for the theme was fun, but was very time-consuming.

Process

Building this particular website was very fun. Not only was planning a Cinco de Mayo party interesting but the colors and design features I implemented allowed me to have creative freedom. Before I began coding, I created a mockup that would better prepare me for the desired result. By being able to visualize the end goal, it created a guide for me to know what to design for which made the process much easier and quicker.
